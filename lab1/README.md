# Задание 3

## Задача 1

### Текст задачи

Создайте собственные функции для выполнения основных операций над списками (добавление/ удаление/поиск элемента, сцепка двух списков, получение элемента по номеру).

### Алгоритм решения

#### Выбор действия(удаление элемента по индексу или сцепка двух списков или получение элемента по индексу)
  ##### Удаление элемента по индексу
    Создание списка
    Получение индекса элемента
    Создание нового списка, без элемента указанного индекса
    Вывод результата
  ##### Сцепка двух списков
    Создание первого списка
    Создание второго списка
    Сцепка списков
    Вывод результата
  ##### Получение элемента по индексу
    Создание списка
    Получение индекса элемента
    Поиск элемента по индексу
    Вывод результата
